    Для более быстрой разработки проектов решил перейти на IDE phpstorm</a>, ранее использовал notepad++, как и большинство разработчиков под битрикс. Оказывается зря, phpstorm позволяет автоматизировать такие рутинные действия как:

    1. Расстановка корректной лесенки для кода</li>
    2. Расстановка пробелов до и после { в функциях и css ( З.Ы. да, это уже болезнь :) )</li>
    3. Подстановка компонентов битрикса</li>
    4. API битрикса в IDE не залезая на сайт</li>
    5. Автоподстановка API</li>
    6. ГИТ интеграция.</li>
      </ol>
      И так, для интеграции phpstorm в проект потребуется следующее:

      1. Создаем новый проект, жамкая клавишу «Create new project from Existing Files»</li>
      2. Выбираем, где располагается проект. Выбираем из 2, 3, 4 пунктов, ни в коем случае не в папке проекта, потому что, иначе, папки bitrix и upload будут индексироваться phpstorm и он будет ужасно тормозить.
