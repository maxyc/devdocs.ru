В этой статье собраны некоторые, на мой взгляд, интересные и полезные запросы в MySQL. Список будет пополняться в будущем.

Выбор записей с повторяющимся значением поля N раз. Замените в запросе N на конкретное число.</h2>

SELECT *

FROM `table_name`

GROUP BY `column_name`

HAVING COUNT(*) = N;</pre>


Событие, которое срабатывает 1 раз в час и удаляет записи из таблицы.</h2>


SET GLOBAL `event_scheduler` = ON;

 CREATE EVENT `hourly_event`

 ON SCHEDULE EVERY 1 HOUR

 DO

 DELETE FROM `table_name`;</pre>


Получение название поля, которое является первичным ключом в таблице.</h2>


SELECT `COLUMN_NAME`

 FROM `information_schema`.`COLUMNS`

 WHERE `TABLE_NAME` = 'table_name'

 AND `COLUMN_KEY` = 'PRI';
