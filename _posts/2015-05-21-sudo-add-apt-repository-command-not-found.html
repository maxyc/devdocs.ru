Иногда бывает так, что на сервере может быть установлена настолько «обрезанная» версия дистрибутива, что стандартых команд в нем просто нет. Например, такие сборки предосталвяют ребята из Locum, ubuntu и debian(остальные и не пробовал) они предоставляют максимально почиканную, в которой нет даже команды add-apt-repository.

Но, на локумовских серверах не так все просто. Сервер под Ubuntu всегда будет кричать, что команды такой нет, и знать я тебя не знаю. Чтобы обойти эту проблему вот что надо сделать.

sudo apt-get install python-software-properties

cd /tmp

wget http://xaxatyxa.ru/download/add-apt-repository.sh.txt

sudo mv add-apt-repository.sh.txt /usr/sbin/add-apt-repository

sudo chmod o+x /usr/sbin/add-apt-repository

sudo chown root:root /usr/sbin/add-apt-repository
