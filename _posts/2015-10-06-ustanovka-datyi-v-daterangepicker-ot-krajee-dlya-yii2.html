Настройка самого виджета http://demos.krajee.com/date-range трудностей не преставляет:

field($conflict, 'dates_worked',[

        'addon' => [

            'append' => [

                'content' => ''

            ]

        ],

    ])->widget(DateRangePicker::classname(), [

        'useWithAddon' => true,

        'convertFormat' => true,

        'pluginOptions' => [

            'format' => 'Y-m-d',

            'separator' => ' / ',

            'opens' => 'left',

        ]

    ]) ?>



Вопрос возник, когда понадобилось программно изменить значения начала и окончания. Решение следующее:

var date = '2014-09-01 / 2014-10-01',

    dates = date.split(' / '),

    daterangepicker = $('#conflict-dates_worked')

                        .closest('.input-group')

                        .data('daterangepicker'); // достаем сам объект daterangepicker

$('#conflict-dates_worked').val(date);  // отображаем новую дату в input-e

daterangepicker.setStartDate(dates[0]); // задаем старт

daterangepicker.setEndDate(dates[1]);   // и окончание в самом календаре
