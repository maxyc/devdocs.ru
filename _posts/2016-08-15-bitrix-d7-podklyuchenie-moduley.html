Как известно при разработке на Битрикс программист должен сам заботиться о подключении необходимых модулей с помощью метода CModule::IncludeModule(). Причем подключать модули необходимо везде, где происходит обращение к api этих модулей. В каждом компоненте, шаблоне, скрипте, функции, методе.

Теперь, вместо старого доброго CModule для подключения модулей нужно применять новый бодрый \Bitrix\Main\Loader.

// Old school
CModule::IncludeModule("iblock");

// D7
use Bitrix\Main\Loader;
Loader::includeModule("iblock");
