Доброго времени суток, всем! Кто-то умный, не помню в какой статье именно на Хабре, буквально недавно размышлял о процессе разработки с явным призывом автоматизировать все, что только можно автоматизировать. И лучше один раз потратить время на автоматизацию, чтобы потом экономить его на протяжении всего проекта.

У веб-разработчиков есть прекрасный инструмент для автоматизации массы задач, который называется Gulp. И моя страсть к таксономии заставила себя собрать огромную коллекцию почти из 100 ценных плагинов для этого сборщика. Думаю многие кто уже использует Gulp найдут для себя что-то нужное, а кто нет, глядя на возможности, получит хорошую мотивацию установить его и понять как эта штука работает.

А еще я выложил подборку на GitHub</a>, чтобы каждый мог пополнить коллекцию.
</a>

HTML&CSS</h4>

autoprefixer</a> — один из самых полезных плагинов, который автоматически расставляет префиксы к CSS свойствам, исходя из статистики caniuse</a>. Важно сказать, что Автопрефиксер это лишь один из множества дополнений в рамках проекта PostCSS</a> от Злых Марсиан.

gulp-browser-sync</a> — вероятно, самый нужный инструмент, с точки зрения повышения продуктивности веб-разработчиков. BrowserSync создает подключение, после чего производит автообновление страницы во всех браузерах на всех устройствах при изменениями не только клиентских или даже серверных файлов. А плюс ко всему синхронизирует позицию скроллинга и заполненные данные в формах.

gulp-useref</a> — парсит специфичные блоки и конкатенирует описанные в них стили и скрипты.

gulp-email-design</a> — бесценный инструмент при верстки писем, который переводит все CSS стили в инлайновые, автоматически изменяя все пути к файлам, опционально умеет загружать изображения на CDN и даже отсылать письма на почту.

gulp-uncss</a> — лучшее решение для оптимизации CSS файлов. Плагин анализирует HTML код и находит все неиспользуемые и продублированные стили.

gulp-csso</a> — отличный CSS минификатор. На сегодняшний день существует целый ряд CSS компрессоров и сравнительная таблица (GitHub) по ним. Но недавно я увидел лучшую Shorthand</a> утилиту без Gulp, которая делает приблизительно следующее:

a</span> {

    font-family</span>: Arial</span></span>;

    font-style</span>: italic</span></span>;

    font-size</span>: 14px</span></span></span>;

    line-height</span>: 18px</span></span></span>;

    font-weight</span>: bold</span></span>;

    background-image</span>: url</span>('example.png'</span>)</span></span>;

    background-color</span>: red</span></span>;

    background-size</span>: cover</span></span>;

    background-repeat</span>: no-repeat</span></span>;

}</span>


=>


a</span> {

  font</span>: italic bold 14px</span>/18px</span> Arial</span></span>;

  background</span>: red url</span>('example.png'</span>) no-repeat / cover</span></span>;

}</span>



gulp-csscomb</a> — облагораживает структуру ваших CSS.


gulp-csslint</a> — CSS линтер.


gulp-htmlhint</a> — HTML валидатор.</blockquote>


JavaScript</h4>


gulp-autopolyfiller</a> — мега крутой плагин, который похож на Autoprefixer и подбирает все необходимые полифилы для JavaScript, чтобы вы могли использовать последние стандарты ECMAScript уже сегодня. Еще рекомендую gulp-babel</a>, который конвертирует ES6/ES7 в ES5.


gulp-jsfmt</a> — полезнейший плагин для работы с JavaScript от команды Rdio, который позволяет искать конкретные фрагменты, форматировать и производить массовые изменения в коде. Также существует gulp-jsbeautifier</a>.


gulp-jscs</a> — JavaScript Code Style. Замечательный инструмент со множеством конфигураций для проверки вашего кода в соответствии с существующими стайлгайдами от jQuery, Яндекса, Google, Airbnb и других.


gulp-modernizr</a> — помогает составить правильную архитектуру проекта на основе Modernizr.js в зависимости от возможностей браузера.


gulp-express</a> — запускает Express.js веб-сервер.


gulp-requirejs</a> и gulp-browserify</a> — оптимизируют работу RequireJS и Browserify соответственно.


gulp-plato</a> — предоставляет аналитику по вашему коду с разными метриками в виде красивых графиков.


gulp-complexity</a> — проверка на качество кода основанная на алгоритмах Halstead и Cyclomatic.


fixmyjs</a> — автоматически исправляет простые ошибки в коде после линта выполненного на основе JSHint</a> (gulp-jshint</a>).


gulp-jscpd</a> — для поиска дубликатов в коде.


gulp-jsonlint</a> — валидатор JSON файлов.


gulp-uglify</a> — JavaScript компрессор.


gulp-concat</a> — конкатенация файлов.</blockquote>

    Графика</h4>
    

    gulpicon</a> — ценный плагин, который позволяет генерировать спрайты из SVG, переводить их в PNG, записывать все Data URI и подключать нужный формат в зависимости от возможностей браузера.
    

    gulp-iconfont</a> — великолепный плагин для работы с веб-шрифтами. Умеет создавать WOFF, WOFF2, EOT, TTF файлы из SVG.
    

    gulp-responsive</a> — простой способ сгенерировать адаптивные изображения под требуемые разрешения устройств с указанием соответствующих префиксов в наименовании.
    

    gulp-sharp</a> — самый быстрый модуль для работы с JPEG, PNG, WebP и TIFF изображениями. Плагин умеет изменять размер, ориентацию, фон, альфа-канал и многое другое.
    

    gulp-svgstore</a> — объединяет все подключаемые SVG файлы и записывает их в HTML как </code> для дальнейшего использования.
    

    gulp-imacss</a> — очень удобная утилита, которая автоматически преобразовывает подключенные в CSS изображения PNG, JPG, SVG в Data URI.
    

    gulp-imagemin</a> и gulp-tinypng</a> для сжатия изображений.
    

    gulp-spritesmith</a> — автоматическая генерация спрайтов.</blockquote>
    

    Разное</h4>
    

    gulp-grunt</a> — позволяет запускать Grunt плагины для Gulp.</blockquote>
    

    Некоторые из отсутствующих:</b></h5>
    

    grunt-html-build</a> — универсальный помощник в верстке. Плагин умеет объединять исходные файлы и создавать полноценные шаблоны/блоки для вашего HTML. Еще для похожих задач очень хорош grunt-include-replace</a>.
    
    grunt-penthouse</a> и grunt-criticalcss</a> — автоматически находят Critical Path в вашем проекте. Важный момент с точки зрения производительность, о чем подробнее написано тут</a>.
    
    grunt-shipit</a> — автоматизирует deploy с помощью Shipit.
    
    grunt-githooks</a> — привязывает Git Hooks</a> к Grunt таскам.
    
    grunt-gitbook</a> — создавайте документацию с помощью потрясающей утилиты GitBook.
    
    grunt-conventional-changelog</a> — генерирует список изменений на основе коммитов в Git.
    
    grunt-proxy</a> and grunt-connect-proxy</a> — connect support for proxying API calls during development.
    
    grunt-phantomas</a> — превосходный инструмент для измерения производительности проекта.
    
    grunt-usebanner</a> — добавляет баннеры (копирайт или ASCII заголовки) в исходных файлах и grunt-figlet</a> для помощи работы с этим.
    
    gulp-watch</a> — краеугольный камень в плагинной системе Gulp. Следит за всеми указанными файлами или целыми директориями и в случае каких-либо изменений выполняет описанные в конфигурациях таски.
    
    gulp-notify</a> — выводит ошибки при сборке Gulp в виде системных сообщений, а главное то, что работает для разных операционных систем.
    
    gulp-git</a> — позволяет использовать Git команды.
    
    gulp-jsdoc</a> — генератор документации, работает на основе JSDoc3.
    
    gulp-rev</a> — полезный плагин для работы с ревизиями.
    
gulp-release — позволяет управлять версиями в вашем проекте.
    
    gulp-bump</a> — следит за репозиторием и обновляет package.json и gulp-update</a>, который обновляет сами пакеты.
    
    gulp-bower-files</a> — подключает все необходимые Bower компоненты.
    
    gulp-removelogs</a> — автоматически удаляет логи.
    
    gulp-preprocess</a> — препроцессор, ссылающийся на установленные конфигурации.
    
    gulp-duration</a> — отображает время выполнения тасков.
    
    gulp-changed</a> и gulp-newer</a> — запускают таски только для изменившихся файлов.
    
    gulp-connect</a> — простой веб-сервер для статических сайтов.
    
    gulp-shell</a> — позволяет запускать Shell команды.
    
    gulp-ssh</a> — обеспечивает возможность подключения по SSH и SFTP.
    
    gulp-zip</a> — архивирует папки и файлы.
    
    gulp-clean</a> и gulp-copy</a> — соответственно очищают и копируют указанные исходники.
    
    gulp-filesize</a> — отображает размеры файлов в удобном для чтения формате.
    
    gulp-util</a> — различные утилиты для разработки Gulp плагинов.</blockquote>
